# Estudos Praticos de javaScript

## 1. Desafio de notas escolares

### Transformar notas escolares

- Crie um algoritimo que transfoorme as notas do sistema numérico para
  o sistema de notas em caracteres.
- Ex.: A B C

* de 90 para cima - A
* entre 80 - 89 - B
* entre 70 - 79 - C
* entre 60 - 69 - D
* menor que 60 - F
